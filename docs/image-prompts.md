## 🎨 Usage Tips

- **DALL-E 3**: Use prompts as-is; works well with detailed scene descriptions
- **Midjourney**: Append `--ar 16:9` or `--ar 1:1` and add style flags like `--style raw` for less stylized results
- **Stable Diffusion XL**: Use with a negative prompt like `"text, watermark, logo, blurry, low quality"` for cleaner results
- **Brand colors**: Primary purple `#512BD4` (.NET), accent orange for HuggingFace elements
- **Consistency**: Use the same seed/style across a series for visual cohesion in blog post sets
